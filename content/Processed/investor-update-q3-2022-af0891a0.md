# Research & Data üî¨üß¨

*   Dimensionality reduction
    *   Our data scientist ran an initial dimensionality reduction experiment to understand how we can best leverage complex microbial data for disease prediction and classification models.
        *   One V1 model was able to predict whether someone was in menopause or not with 85% accuracy.
    *   While there is a lot more to do here, this is exciting initial validation that the information Evvy is collecting via metagenomic sequencing is valuable for accurate disease prediction.
*   MAG (metagenome assembled genome) Pipeline
    *   We have been able to generate high quality genomes from our metagenomic data. The pipeline has been optimized and is currently running on all 6000 samples. The output of this will be assembled microbial genome data from individual samples.
    *   This data will be used for biomarker discovery outside of traditional taxonomic classification. This data will allow for novel analysis into sub-species variation, genetic content comparisons, and even in/del and SNP identification.
*   SAGE database completed
    *   The Select annotated genome (SAGE) database has been compiled from 4315 genomes representing 712 species isolated specifically from the urogenital and vaginal tract.
        *   For comparison, the publicly available Virgo database only contains 264 species
    *   Evvy's proprietary reference genome database is being implemented in the clinical version of our test to provide more accurate taxonomic classification.
*   First publication accepted to [mSystems](https://journals.asm.org/journal/msystems) Journal in collaboration with UVA Computational Systems Biology Lab
    *   Title: *Metabolic network models of the Gardnerella pangenome identify key interactions in the vaginal environment*
    *   mSystems is part of the American Society for Microbiology Papers with a high impact factor
