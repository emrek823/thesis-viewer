## STATEMENT OF WORK

Licensed Avance Clinical users shall be granted access to the secure, private Ryght platform for general use upon contracting and kick-off.

Ryght shall develop a series of copilots for Avance Clinical (see example list below). Ryght shall not charge Avance Clinical for the development of these copilots. However, use cases that require a significant amount of engineering resources shall be discussed on a case by case. If Avance Clinical requests, or requires, the fine tuning of a machine learning (ML) or large language model (LLM), those fees shall be negotiated separately. Avance Clinical would own the intellectual property of the aforementioned fine tuned LLM.

Ryght will initially undertake the Participant information consent form generator

**Protocol Content Generation copilot:**

*   Scope end-to-end user requirements and copilot interface
    *   Map current Avance proposal process
    *   Highlight protocol sections, trial type, therapeutic areas etc. as steps
    *   Map protocol sections to relevant Avance data
    *   Identify example questions from a user for prompt engineering
*   Data integration:
    *   Discover necessary data sources for interpolation
    *   Determine secure methods for connecting sources
    *   If necessary, develop connectors
    *   Structure data for access
*   Develop orchestration process, prompts and copilot interface
*   Outline testing process and acceptance criteria
    *   Confirm response content quality with users or stakeholders
*   Setup deployment process for iterative development
*   Ryght resources: solution architect, AI / data engineer, customer success specialist
*   Avance Clinical resources: Anne Sim or designee; any IT resources for data integration
*   Target project kick-off date: **February, 2024**
