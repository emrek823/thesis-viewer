### Mare Insights Product Overview

- Healthcare data analytics platform for sales teams

	- Traditional implementations: 6-8 weeks minimum
	- Mare solution: 60 minutes turnaround
- Core problem: Point solution vendors to payers/providers struggle with ROI storytelling and deal scoping
	- Need client data access to demonstrate value

### Platform Architecture & Capabilities

- Argo portal for prospect self-service data upload
	- Supports medical claims, eligibility, pharmacy files
	- Deidentified data processing
	- Intuitive file type differentiation
- Data quality grading system (A-F scale)
	- A: All required fields present, good quality
	- B: Required fields present, some improvement needed
	- F: Missing required fields, inadequate for analysis
- Backend processing through Tuva integration
	- Full Tuva database implementation
	- Custom denormalized data marts for ROI algorithms
	- Standard extract formats for client consumption
