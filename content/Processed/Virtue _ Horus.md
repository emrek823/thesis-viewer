- Technical differentiation:
	- Orchestration layer with sub-agents for record review
	- Consensus-based reasoning mechanism
	- Hierarchical code search through ICD/CPT decision trees (vs. searching all 160,000 codes)
	- Rule-based NLP first, then LLM contextual reasoning
- Benchmarking approach:
	- Using 3M gold standard dataset identifying missed codes in Mimic 3 ground truth
	- Optimizing for delta between gold standard and ground truth
