## Slide 7: Orchestra
# Orchestra
Our managed SaaS platform utilizes machine learning to tie together
data, models, and workflow across the enterprise

| Stage     | Component                                      | Description                                                                  |
| :-------- | :--------------------------------------------- | :--------------------------------------------------------------------------- |
| **DATA**  | **Data connectivity, harmonization, storage, governance, etc...** | e.g. Ingest payer, banking, contract, and operational data, and harmonize it the appropriate schema |
| **MODEL** | **ML models or heuristics that act on core data** | e.g. Identify and flag mismatches vs. expected                               |
| **WORKFLOW** | **Write back to core systems, generate alerts, etc...** | e.g. Update core systems based on workflow-driven actions                    |
