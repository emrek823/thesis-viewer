# Slide 5: Using Ryght to extract data

Using Ryght to extract data

- Ryght's LLM pipelines work well with textual information
  and **context**
- This can be exploited to pull target data from a block of
  text, regardless of where it's placed or what it is
- Feed Ryght a report as context, and instruct it to structure
  the data into tables
- Able to infer relationships even when data is spread far
  apart in the text
